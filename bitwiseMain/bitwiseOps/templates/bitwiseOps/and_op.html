The AND (&) bitwise operation is used to "clear" a bit,
bit clearing is changing a specific bit in a number to 0
without affecting the other bits.
                

Example:

Number in 8 bits:   00000111   (7 in decimal)
Goal in 8 bits:     00000101   (5 in decimal)

To turn binary 7 into binary 5 we need to clear its second bit

00000111

To do this we need a bit mask that looks like this:

11111101

To obtain this mask we will LEFT SHIFT 1 by two positions (1<<2):
this will give us this bit mask:

00000010

now we need to invert these bits by using NOT ~(1<<2)
this gives us the bit mask we need 

11111101 

and we can use AND to clear the second bit of 7:

7 & ~(1<<2)

This is what it looks like in C:

#include "stdio.h"

    int main() {
        uint8_t num = 7;   // Binary: 00000111
        num = num & ~(1<<2);  // 00000111 & 11111101
        printf("New number: %d\n", num); // Output: New number: 5
        return 0;
    }
