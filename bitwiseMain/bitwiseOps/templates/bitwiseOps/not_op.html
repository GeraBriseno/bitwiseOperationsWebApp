The NOT (~) bitwise operation is used to "flip" a bit, bit flipping will change all bits in a number to their opposite (0 to 1, 1 to 0).
                

Example:

Number in 8 bits:   00000111   (7 in decimal)

We will be flipping all the bits in the number 7

with the NOT operation: ~(7) will give us 11111000
that is 248 in decimal

This is what it looks like in C:


#include "stdio.h"

    int main() {
        uint8_t num = 4;   // Binary: 00000111

        num = ~(num);  // 11111000
        printf("New number: %d\n", num); // Output: New number: 248
        return 0;
    }
