Bitwise Operations Explained

                    Bitwise operations in C are used to directly manipulate the bits of a number. In computers, numbers are stored in binary (0s and 1s). Bitwise operators let you work with these bits. Here are the main bitwise operators:
                

                        AND (&)
                        Compares each bit of two numbers.
                        The result is 1 only if both bits are 1. Otherwise, it's 0.
                    
                        OR ( | )
                        Compares each bit of two numbers.
                        The result is 1 if at least one of the bits is 1.
                    
                        XOR (^)
                        Compares each bit of two numbers.
                        The result is 1 only if the bits are different.
                    
                        NOT (~)
                        Flips all the bits.
                        1 becomes 0, and 0 becomes 1.
                    
                        LEFT SHIFT (<<)
                        Moves bits to the left by a certain number of positions.
                        Each shift to the left multiplies the number by 2.
                    
                        RIGHT SHIFT (>>)
                        Moves bits to the right by a certain number of positions.
                        Each shift to the right divides the number by 2.
