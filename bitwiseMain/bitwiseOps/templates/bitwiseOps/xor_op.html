XOR (^)

                    The XOR (^) bitwise operation is used to "toggle" a bit,
                    bit toggling is changing a specific bit in a number
                    to 0 if it is 1 or to 1 if it is 0 without affecting the other bits.
                

                        Example:

                        Number:   00000100   (4 in decimal) 

                        We will be toggling the third bit in the number 4

                        00000100
                             ^
                            
                        number 4 looks like this using LEFT SHIFT (1<<3)
                        
                        the XOR operation: 4 ^ (1<<3) will give us 4 or 0, depending
                        on its previous value

                        This is what it looks like in C:

                        #include "stdio.h"

                            int main() {
                                int num = 4;   // Binary: 00000100

                                num = num ^ (1 << 3);  // 00000100 ^ 00000100
                                printf("New number: %d\n", num); // Output: New number: 0

                                num = num ^ (1 << 3);  // 00000000 ^ 00000100
                                printf("New number: %d\n", num); // Output: New number: 4
                                return 0;
                            }
